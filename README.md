# OverTheWire Bandit Writeups

Welcome to my OverTheWire Bandit learning repository.

This repository documents my journey through the Bandit wargame while learning Linux commands, terminal usage, and cybersecurity fundamentals.

* Learn Linux from the basics
* Understand terminal commands
* Improve problem-solving skills
* Build a strong cybersecurity foundation
* Document everything I learn

* Objective of the level
* Commands used
* Explanation of each command
* Solution approach
* What I learned
* Key takeaways

> Passwords are intentionally **not included** to respect the OverTheWire challenge.
